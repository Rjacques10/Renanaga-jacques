## 📌 À propos

Ce site présente mon parcours, mes compétences et mes projets en cybersécurité et développement web. Il est construit en HTML/CSS/JS pur (aucun framework), pensé pour être rapide, léger et facilement personnalisable.

**Points clés :**
- 🌗 Thème clair / sombre
- 🇫🇷 🇬🇧 Bilingue français / anglais (bouton de bascule)
- 🖥️ Terminal animé en page d'accueil (effet `whoami`)
- 🌍 Globe animé en fond (D3.js)
- 🔒 Distinction claire entre projets publics et privés
- 📚 Récapitulatif complet du cursus (Niveau 100 → Niveau 300, en cours)

## 🚀 Voir le site en local

Aucune installation nécessaire — c'est un simple fichier statique.

```bash
git clone https://github.com/Rjacques10/Porfolio.git
cd Porfolio
# puis ouvre index.html directement dans ton navigateur
```

## 🌐 Déploiement (GitHub Pages)

1. **Settings** du dépôt → **Pages**
2. Source : `Deploy from a branch`
3. Branche : `main`, dossier : `/ (root)`
4. Sauvegarder — le site est en ligne après 1-2 minutes à l'adresse fournie par GitHub

## 🛠️ Stack technique

| Élément | Détail |
|---|---|
| Structure | HTML5 |
| Style | CSS3 (variables custom, thèmes clair/sombre) |
| Interactivité | JavaScript vanilla |
| Polices | JetBrains Mono, Inter (Google Fonts) |
| Visualisation | D3.js (globe animé) |
| Hébergement | GitHub Pages |

## ✏️ Personnaliser

Le contenu texte est centralisé dans le dictionnaire `translations` (objets `fr` et `en`) en bas du fichier `index.html` : modifie une valeur là-bas, elle se répercute partout où la clé `data-i18n` correspondante est utilisée.
